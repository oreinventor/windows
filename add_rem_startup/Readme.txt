Toggle StartUp on right click context
	Created by Thgo, oreinventor, iNi

Description:

	This bat script should allow/remove a program from auto start on the windows startup

Instructions:

	1- Copy add_rem_startup.bat to C:\
	2- Open reg file
	3- Right click on desired program and click on ToggleStartUp
	The desired program should start/not start from the next startup
